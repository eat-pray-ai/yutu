# Yutu SuperChatEvent

## Overview

This skill allows you to manage YouTube Super Chat events using the `yutu` CLI tool. You can retrieve a list of Super Chat events associated with your channel.

## SuperChatEvent Operations

### List Super Chat Events

Retrieve a list of Super Chat events.

**Reference:** [references/superChatEvent-list.md](references/superChatEvent-list.md)

**Common Tasks:**

- List events: `yutu superChatEvent list`
