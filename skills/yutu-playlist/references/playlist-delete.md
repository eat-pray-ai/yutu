# Playlist Delete Command

Delete a playlists by ids.

## Usage

```bash
yutu playlist delete [flags]
```

## Flags

| Flag | Shorthand | Description |
|------|-----------|-------------|
| `--ids` | `-i` | IDs of the playlists to delete |
| `--onBehalfOfContentOwner` | `-b` | |

## Examples

```bash
# Delete a playlist
yutu playlist delete --ids PLAYLIST_ID
```
