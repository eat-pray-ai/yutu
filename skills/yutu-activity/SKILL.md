# Yutu Activity

## Overview

This skill allows you to manage YouTube activities using the `yutu` CLI tool.

## Activity Operations

### List activities

List activities. Use this tool when you need to list activities such as uploads, likes, and favorites.

```bash
# List my activities
yutu activity list --mine
```
