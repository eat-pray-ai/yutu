# Watermark Unset

Unset a watermark for channel's videos. Use this skill to unset a watermark for a channel's videos.

## Usage

```bash
yutu watermark unset [flags]
```

## Flags

| Flag | Shorthand | Required | Description |
|------|-----------|----------|-------------|
| `--channelId` | `-c` | Yes | ID of the channel |

## Examples

```bash
# Unset watermark for a channel
yutu watermark unset --channelId UC_x5XG1OV2P6uZZ5FSM9Ttw
```
