# Watermark Unset Command

Unset watermark for channel's video by channel id.

## Usage

```bash
yutu watermark unset [flags]
```

## Flags

| Flag | Shorthand | Description |
|------|-----------|-------------|
| `--channelId` | `-c` | ID of the channel |

## Examples

```bash
# Remove watermark
yutu watermark unset
```
