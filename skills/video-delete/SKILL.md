# Video Delete

This skill provides instructions for deleting YouTube videos using the `yutu` CLI.

## Usage

```bash
yutu video delete [flags]
```

## Options

- `--ids`, `-i`: IDs of the videos to delete.

## Examples

**Delete a video:**

```bash
yutu video delete --ids VIDEO_ID
```
