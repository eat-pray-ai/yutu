# Video Rate Command

Rate a video on YouTube.

## Usage

```bash
yutu video rate [flags]
```

## Flags

| Flag | Shorthand | Description |
|------|-----------|-------------|
| `--ids` | `-i` | IDs of the videos to rate |
| `--rating` | `-r` | like\|dislike\|none |

## Examples

```bash
# Like a video
yutu video rate --ids VIDEO_ID --rating like
```
