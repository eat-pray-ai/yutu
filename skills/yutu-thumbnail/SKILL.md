# Yutu Thumbnail

## Overview

This skill allows you to manage YouTube video thumbnails using the `yutu` CLI tool. You can upload and set custom thumbnails for your videos.

## Thumbnail Operations

### Set Thumbnail

Upload and set a thumbnail for a video.

**Reference:** [references/thumbnail-set.md](references/thumbnail-set.md)

**Common Tasks:**

- Set thumbnail: `yutu thumbnail set --videoId VIDEO_ID --file image.png`
