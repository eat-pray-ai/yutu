# Comment Delete

This skill provides instructions for deleting YouTube comments using the `yutu` CLI.

## Usage

```bash
yutu comment delete [flags]
```

## Options

- `--ids`, `-i`: IDs of the comments to delete.

## Examples

**Delete a comment:**

```bash
yutu comment delete --ids COMMENT_ID
```
