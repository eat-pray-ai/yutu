# Yutu VideoCategory

## Overview

This skill allows you to manage YouTube video categories using the `yutu` CLI tool. You can retrieve a list of video categories available in different regions.

## VideoCategory Operations

### List Video Categories

Retrieve a list of video categories.

**Reference:** [references/videoCategory-list.md](references/videoCategory-list.md)

**Common Tasks:**

- List categories: `yutu videoCategory list --regionCode US`
